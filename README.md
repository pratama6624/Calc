# Swift Simple Calculator & Converter

A simple iOS app built with **SwiftUI** and **MVVM**.  
Includes:
- Standard Calculator  
- Binary Calculator
- BMI Calculator
- Financial Calculator
    - Currency Calculator
    - Loan
    - Compound Interest
    - Mortgage
    - ROI
    - Profit Margin
    - Break-Even
    - Savings Growth
- Unit Converter  
    - Pressure
    - Volume
    - Weight
    - Speed
    - Time
    - Temperature

Built for practicing **UI design** and **logic separation** in Swift.
