# Swift MVVM Calculator & Converter

A simple iOS app built with **SwiftUI** and **MVVM**.  
Includes:
- Standard Calculator  
- Currency Converter
- Binary Calculator
- BMI Calculator
- Unit Converter  

Built for practicing **UI design** and **logic separation** in Swift.
